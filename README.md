### PathTracerCUDA
 
# Description

An interactive GPU accelerated path tracer with support for diffuse, specular and metallic surfaces. Path tracing algorithm uses physically plausible Phong BRDF importance sampling to determine the next direction on the path [PBRPhong](https://www.cs.princeton.edu/courses/archive/fall03/cs526/papers/lafortune94.pdf).
